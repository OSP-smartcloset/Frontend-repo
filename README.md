# Smartcloset
SW 개발자 공모전 대회
개발기간: 2024.07 ~ 2024.08

## 팀원 소개
백엔드 - 안현준, 채린, 명우성, 최재혁
프론트엔드 - 신은섭

사용자의 위치, 날씨 그리고 상황에 따른 코디를 생성형 AI를 통해 제안해주는 서비스이다.
